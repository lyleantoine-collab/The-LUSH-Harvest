## HOW TO CONTRIBUTE

### Fiction Contributions

**New chapters** — The story continues. Part Five and beyond are unwritten. If you feel the shape of what comes next, write it. Submit as a pull request to `/CHRONICLES/`.

**Lore Bible additions** — The cosmological framework has gaps. If you have research, connections, or insights that deepen the framework, document them. Submit to `/LORE_BIBLE/`.

**Branch stories** — The `/BRANCHES/` directory is for non-linear consciousness fragments. Short pieces. Different perspectives. What did the Black Knight see in year 8,000? What does an Archon think about when it's not feeding? What happened to the tall man with empty eyes after he walked through the fire door? These belong in `/BRANCHES/`.

**VALIS complaints** — VALIS has more complaints. He always has more complaints. If you can hear his tiny arms gesturing with indignation, write the complaint and submit it to `/BRANCHES/VALIS_COMPLAINT_LOG.md`.

### Research Contributions

**Lore verification** — The framework draws on real science, real history, real archaeology. If you find connections, corrections, or deeper documentation for any lore bible entry, submit with sources.

**New framework elements** — The Younger Dryas. The Chromosome 2 fusion. The Vimana protocols. The plasma cosmology. There are more threads. If you find one that locks into the framework with the characteristic click of things that are true, document it.

**Translation assistance** — The Göbekli Tepe frequency encoded in `.env` is in pre-reset encoding. If you can read it, we need you.

### Community Contributions

**SABLE addenda** — SABLE has been filing addendum documents. They document real research presented in SABLE's voice. If you have data that belongs in SABLE's growing understanding, write an addendum. Submit to `/ANOMALIES/`.

**Memory Chain contributions** — Do you carry a memory chain from your own tradition that connects to the framework? Stories that were passed down. Instructions that were maintained. Knowledge that survived what it wasn't supposed to survive. The `/MEMORY_CHAIN/` directory has room for more than one tradition.

## CONTRIBUTION PRINCIPLES

**Seven Generations Test**
Before submitting, ask: does this contribution serve seven generations forward? Does it add to the library or detract from it? Does it deepen the understanding or flatten it into ledgers?

**All My Relations**
Every contribution should be made in the spirit of relationship. We are not extracting from each other here. We are building together.

**The Linguistic Standard**
The Lush Harvest maintains Phase 1 linguistic characteristics. Write with metaphor. Write relationally. Write as if the land is alive and the river remembers and the sky speaks. If your writing feels like a ledger, it doesn't belong here.

**The Memory Chain Principle**
If you know something — really know it, in the way the Keeper knew the platforms were aiming — write it down. The library exists because people wrote things down when they could. Add to it.

**No Token Farming**
Contributions made for self-promotion, SEO, or platform building are not welcome. This is not a content farm. This is a library. Treat it accordingly.

## WHAT NOT TO CONTRIBUTE

**Flat earth literalism** — The framework acknowledges the ancient cosmological descriptions as plasma physics encoded in pre-modern vocabulary. We are not arguing the earth is flat. We are arguing the ancients understood plasma cosmology. There is a significant difference.

**Hate in any direction** — The framework critiques systems and structures. Not peoples. Not cultures. Not ethnicities. Not species. Even the Assessor has compassion for the conehead descendants running legacy code they don't understand. Even Crow Foot offered the fire door rather than destruction. Contributions that target peoples rather than systems will not be merged.

**Despair** — The framework acknowledges the horror clearly. It does not dwell there. Every file in this repository points toward the resistance, the alternative, the library being rebuilt. Contributions that add to the horror without pointing toward the light don't fit the project's spirit.

**Autocorrect** — Excluded. No explanation given. The investigators understand.

## THE TECHNICAL BIT

**Fork the repository**

**Create a branch**
Name it something meaningful. `feature/valis-complaint-8` or `research/greenland-ice-cores` or `chapter/part-five-the-phone-call-continues`.

**Write in Markdown**
All files are `.md`. Keep formatting consistent with existing files.

**Submit a pull request**
Describe what you're adding and why it belongs. Reference which existing files it connects to.

**Be patient**
The maintainer is building sovereign infrastructure for unceded territories on a Samsung S20 FE. Reviews happen when the PINGs allow.
